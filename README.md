# WEB-DEV-CAPSTONE-PROJECT
# Capstone Project: Learnify - Online Learning Platform

## Theme Chosen
**Online Learning Platform** - A complete landing page featuring course cards, testimonials, pricing plans, and contact form.

## Project Features

### ✅ **Semantic HTML5 Structure**
- Proper use of `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- ARIA labels and roles for accessibility
- Skip to main content link
- Semantic headings hierarchy (h1-h4)

### ✅ **CSS3 Flexbox & Grid Layouts**
- **Flexbox**: Header navigation, hero stats, footer, contact items, testimonial authors
- **CSS Grid**: Courses grid (responsive 1-3 columns), pricing cards, testimonials
- **CSS Variables**: Consistent theming with `:root` custom properties
